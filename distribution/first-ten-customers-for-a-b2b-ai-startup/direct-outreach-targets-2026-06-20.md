## Operating rule

This is not a blast list.

Use the order below:

1. Comment/reply publicly if there is an active related post.
2. Send directly only when there is a specific reason.
3. Track replies and forwards, not just clicks.
4. Stop after 10 sends if there is no signal; rewrite the angle before continuing.
