# Daily operating schedule

Created: 2026-06-22T00:25:09+05:30.

Purpose: daily execution rhythm for the free X + LinkedIn reach campaign around `First ten customers for a B2B AI startup`.

Execution is tracked in `daily-activity-ledger.csv`. Use that ledger to compare planned daily cadence against actual posts, reply batches, comment batches, outreach, metric checks, and verified replies.

The current strategy is not "post more." It is:

1. Keep one strong native idea alive.
2. Enter rooms where the right audience is already paying attention.
3. Turn replies/comments into the next native post.
4. Track replies and profile movement, not just clicks.

## Daily volume targets

### New native posts

X:

- 1 standalone native post per day.
- Use the essay, live replies, and high-performing comments as raw material.
- Do not add the essay link unless it is a follow-up reply or the post has already earned engagement.

LinkedIn:

- 1 new native post every 24-48 hours.
- Target 3-4 LinkedIn posts per week, not multiple posts per day.
- Use LinkedIn posts for bigger narrative moves, not every small thought.

Default rule:

- X tests ideas faster.
- LinkedIn compounds better when the post is more considered and comment activity feeds profile discovery.

### Replies and comments

X:

- 2 reply batches per day.
- 8-12 replies per batch.
- Daily target: 15-25 useful replies.

LinkedIn:

- 2 comment batches per day.
- 5-8 comments per batch.
- Daily target: 10-15 useful comments.

Every reply/comment should be no-link by default and should add one of:

- a sharper frame,
- a useful counterpoint,
- a concrete example,
- a founder/operator question,
- a missing practical caveat.

## Daily cadence

### Morning block

Goal: protect the existing surface area.

Actions:

1. Refresh X root thread, LinkedIn anchor, and any follow-up post metrics.
2. Check X and LinkedIn notifications for real replies.
3. Reply to every campaign-quality inbound within 30-90 minutes.
4. Publish the day's X post only if no higher-leverage reply is waiting.
5. Log metrics in `metrics-log.csv` and meaningful responses in `reply-source-scorecard.csv`.

### Midday block

Goal: enter high-attention rooms while posts are active.

Actions:

1. Scout LinkedIn posts from founders, operators, investors, and GTM/product leaders.
2. Post 5-8 no-link LinkedIn comments only after explicit approval for that batch.
3. Scout X posts with visible discussion around B2B AI, vertical AI, agents, first customers, pilots, founder-led sales, and GTM.
4. Post 8-12 X replies only after explicit approval for that batch.
5. Record target, author, topic, visible counters, and comment/reply text.

### Evening block

Goal: convert signal into the next asset.

Actions:

1. Refresh metrics and notifications.
2. Identify which channel created replies, profile views, follows, or website clicks.
3. Draft tomorrow's X post from the strongest live exchange.
4. Draft the next LinkedIn post only if the current LinkedIn surface has slowed or a strong new angle emerged.
5. Decide whether to request approval for one of:
   - replying to Luke Sophinos,
   - posting the X high-engagement reply batch,
   - posting another LinkedIn high-engagement comment batch,
   - publishing the horizontal GPT follow-up,
   - sending direct outreach batch 2.

## Additional daily work

### Direct outreach

- Target 5-10 specific people per day.
- Send only when the essay is genuinely useful to that person.
- Prioritize people who can reply, forward, or create second-degree distribution.
- Use `direct-outreach-batch-2-messages.md` and `direct-outreach-batch-3-messages.md` only after explicit approval and live target verification.

### Content recycling

Every day, convert at least one useful conversation into a draft:

- an X post,
- a LinkedIn post,
- a future reply/comment,
- or a direct note angle.

Do not recycle weak vanity engagement. Only recycle material that reveals an objection, useful language, or a sharper wedge.

### Metric review

Check 2-3 times per day:

- X thread/post views,
- X replies/reposts/likes,
- LinkedIn impressions,
- LinkedIn profile viewers,
- LinkedIn first-comment impressions,
- website clicks if visible,
- verified replies in `reply-source-scorecard.csv`.

After each day, update `daily-activity-ledger.csv` with planned vs actual activity. If actual execution is below plan for two consecutive days, the next day's priority should be approved replies/comments before any new standalone post.

Do not judge comments by likes alone. Judge them by:

- replies from target people,
- profile-view movement,
- follow-on conversations,
- raw material for better posts.
